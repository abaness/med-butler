## 五、关键路径（Critical Path）

项目关键路径（任一环节延期都会影响最终上线）：

```
立项(W0)
  → PRD 定稿(W2.5)
    → UI 终稿(W3.3)
      → 核心后端接口(W5.5)
        → 前端核心功能(W7.5)
          → 联调完成(W8.5)
            → 测试通过(W9.5)
              → 正式上线(W10.5)
```

**重点关注**：W2.5 PRD 定稿、W6 核心链路联调、W9 Bug 收敛这三个节点最容易卡壳。
