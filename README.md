## 🎯 一句话产品定位

> **美团慢病用药小管家——让慢病用药，不再靠记性。**

一个基于 AI 的慢病用药全周期管理工具，在美团医药体系内打通"问诊—购药—用药—续方"四环闭环。

## 🔑 差异化亮点

1. **订单即计划** —— 美团买药订单自动生成用药计划，零录入门槛
2. **30 分钟极速续方** —— 结合美团骑手网络，用完即送
3. **AI 管家** —— 医嘱结构化、相互作用、漏服补救、对话问答
4. **老人模式 + 家人关怀** —— 超大字体、语音主导、远程代管
5. **闭环设计** —— 问诊、购药、用药、续方一条链路打通
